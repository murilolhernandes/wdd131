Site Name:

The name of the website will be WORLD CUP RUGBY LEGENDS

I chose this name because it is catchy and easy to remember, it also helps the visitors to understand what the website is about.

Site Purpose:

This website will provide the power house nations of the rugby world. It will list the bigger national teams that have made an impact on the last decade of the Rugby World Cup.

It will also contain a table that will list the top 4 champions of the last World Cup.

Scenarios:

Which country won the last World Cup?

Which country will host the next World Cup?

What are the top 4 champions of the last World Cup?

Color Schema:
